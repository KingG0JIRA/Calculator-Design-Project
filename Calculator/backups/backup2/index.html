ASSIGNMENT PROJECT

CALCULATOR

Project Overview:

The project aims to develop a user-friendly and functional scientific calculator. This web-based calculator supports basic arithmetic operations like addition, subtraction, multiplication, and division, as well as more advanced mathematical functions such as square root, trigonometric operations (sin, cos, tan), logarithmic functions (log, ln), and exponential calculations.

Key Features:

  • Arithmetic Operations: Perform basic operations like addition, subtraction, multiplication, and division.
  • Scientific Functions: Supports advanced functions such as trigonometric functions (sin, cos, tan), inverse functions, square root, power (exponentiation), and constants like Pi.
  • UI Design: The user interface is designed to be simple and intuitive, with a clear output display and easy-to-use buttons for calculations.
  • Responsive Design: The design ensures that the calculator works well across different screen sizes, making it accessible on both desktop and mobile devices.
  • Interactive Menu: Includes a sliding side navigation menu for additional features like switching between light and dark themes and accessing the credits.

Technologies Used:

  • HTML: Structure and layout of the web page.
  • CSS: Styling for the calculator, buttons, and layout.
  • JavaScript: Handles the functionality of the calculator, including mathematical operations and interactions with the user interface.
  • Font Awesome & Bootstrap Icons: Used for adding icons to enhance the design and user experience.

Future Enhancements:

  • Adding additional scientific functions like logarithmic bases, factorials, and other advanced operations.
  • Improving the design to include a history log of previous calculations.
  • Providing more themes or skins for better user customization.
